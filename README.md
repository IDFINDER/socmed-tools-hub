## ✨ **ما يقدمه هذا البوت؟**

مركز تحكم واحد يجمع جميع بوتاتي المتخصصة في تحميل وتحليل محتوى منصات التواصل الاجتماعي.

---

## 🎬 **البوتات المتاحة حالياً**

| البوت | المعرف | الوظيفة |
|-------|--------|---------|
| 🎬 بوت استخراج روابط | @YouTube_Playlist_Extractor_bot | استخراج روابط قوائم التشغيل والقنوات |
| 📸 بوت تحميل صور | @YouTube_photos_Extractor_bot | تحميل صور الفيديوهات بأعلى جودة |
| 📊 بوت تحليل بيانات | @YouTube_data_analyzer_bot | تحليل إحصائيات الفيديوهات والقنوات |
